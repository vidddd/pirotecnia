CSS components

Sangoma template is packed with additional pre-styled css classes wich makes it even more usable. We are doing are best to keep our style integrated and up to date with latest Twitter Bootstrap framework distribution.

Here you can find additional styles for some pre-defined Bootstrap CSS components.

Media table

Use .table-media class for prestyled table with thumbnails.

ID Image Description Category Actions

A lot of templates do not support vertical table header cells. We do. You can combine <th> as you need (in terms of W3C specification).

Forms

All form elements are styled according to Twitter Bootstrap syntax. I you are not familiar with bootstrap framework you should check their official documentation.

Sangoma template uses several form handling plugins. Check them out on plugins page.

Icons with Font Awesome

Font Awesome is iconic font designed for use with Twitter Bootstrap with over 220 icons. You can still use original icons by removing Font Awesome from LESS and recompiling it. For more info and updates vist official Font Awesome page.
